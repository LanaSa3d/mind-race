# Mind Race: Quiz Game Web App

Mind Race is a PHP and MySQL quiz game web application inspired by live classroom quiz platforms. Hosts can create quizzes, add questions, start a game using a unique game code, and let players join, answer questions, earn scores, and view a leaderboard.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP
- PDO for database access

## Planned Features

- Quiz creation and management
- Question CRUD operations
- Game code generation
- Player join flow
- Timed quiz play
- Score calculation with response speed bonus
- Leaderboard ranking
- Admin dashboard statistics
- Sample MySQL database with seed data

## How To Run With XAMPP

1. Start Apache and MySQL from XAMPP.
2. Copy the `mind-race` folder into your XAMPP `htdocs` folder.
3. Open phpMyAdmin at `http://localhost/phpmyadmin/`.
4. Create/import the database by importing `database.sql`.
5. Open the website at:

```text
http://localhost/mind-race/
```
