# Mind Race Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete PHP/MySQL quiz game website with CRUD, timed play, scoring, leaderboard, dashboard, README, and seed database.

**Architecture:** Use root-level PHP pages with reusable includes for layout, database access, helpers, and flash messages. Store all quiz, question, player, and answer state in MySQL via PDO prepared statements. Keep JavaScript focused on validation, delete confirmations, and the quiz countdown timer.

**Tech Stack:** HTML5, CSS3, JavaScript, PHP, MySQL, XAMPP, PDO.
